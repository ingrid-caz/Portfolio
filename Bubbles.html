Bubbles, 2025

For this project we had to create a multisensorial experience in four days from scratch, inspired by snoezelen environnments to be used by non-verbal autistic persons.

It mainly relies on the creation of bubbles thanks to a bubble machine as well as homemade lava lamps. The interraction between the audience and the installation had to be a central point in our work: we used buttons to trigger light and sound around the lava lamps, and a pedal on the ground to start the bubble machine.
